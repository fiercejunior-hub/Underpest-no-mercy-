# Underpest No Mercy

An Undertale fangame set in an alternate universe where red flowers infect monsters, transforming them into violent, hungry, zombie-like creatures. In this world, the only option is to spare—the only way to survive is to fight the infected.

## Story
Frisk and Flowey explore the destroyed Underground, finding survivors and fighting the infected. The goal is to reach the castle and collect the six human souls to escape the Underground.

## Battle System
- Inspired by Undertale's turn-based combat.

- Two types of infected:

- Fully infected: cannot be spared; attempting to spare them causes damage.

- Newly infected: can be spared, but return fully infected later, forcing the player to fight.
